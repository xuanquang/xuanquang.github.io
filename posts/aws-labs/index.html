Learning by doing with https://cloudjourney.awsstudygroup.com/

The First Cloud Journey (FCJ) provides over 100 hands-on exercises that are guided by detailed instructions and images (by many authors). We will explore and study with awesome technologies and exciting services, as well as migrate applications to the cloud and apply best practices to improve your systems.

I wrote some labs for FCJ while learning about AWS Cloud. I’m hoping that some labs will be chosen to publish on FCJ.

Lab Transfer

https://transfer-lab.netlify.app/
In this lab, you will use AWS Transfer Family to setup SFTP access to an Amazon S3 bucket in order to manage files in the Amazon S3 bucket.

Lab Amplify

https://amplify-lab.netlify.app/
In this lab, we will create a React application and deploy it to the cloud using the AWS Amplify web hosting service.

Lab Macie

https://macie-lab.netlify.app/

A solution that can monitor your S3 bucket on daily basis and look for specific personally identifiable information.
